This repository contains miscellanous tasks with LLM

#LLM Summarisation 

Generate prompts for LLM API / summarisation
Purpose
Get familiar with an LLM API and basic file processing.

Use OpenRouter API.

10 English articles, approximately 300-1000 words each.
Save the articles as text file(s).
Create a new Google Colab notebook.
Upload the text files to Colab.
Write Python code that reads each file and uses an LLM API to summarize it.
Use a loop to process all articles.
Print 10 summaries on the screen.
Save all summaries as one JSON file.

#Advance Prompt Engineering
Advanced Prompt Engineering with APIs and multi-model comparison
Task
Investigate five diverse LLMs available through OpenRouter. Qwen3-32B, DeepSeek-R1-0528, Llama 3.3 70B Instruct, Ministral 3 3B 2512 and Gemma 3 27B were used in this notebook.

Test each model on five tasks:
Text summarization
Text generation
Question answering
Data generation
Code generation
For each task, compare model outputs and decide which model performed best.
